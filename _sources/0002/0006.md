# 梅茵 作业与读者
````{div}
孤儿院有两层楼。一楼是公共空间，孩子们平时在这里活动。二楼是居住区，十人一个房间，男孩住东侧，女孩住西侧，灰衣修女带着四岁以下的孩子住在中间。玛奥不住孤儿院，他和灰衣修士一起住在田地边的屋子里。  


梅茵对这里的生活很满意。食物可口，卫生条件也很好。不仅如此，孤儿院的自来水是神力驱动的，梅茵用起来一点心理负担都没有。  


珂琳娜家也有自来水，但需要人力提水维持，梅茵不太愿意用，因为这会让她想起提水的托莉。刚占据梅茵身体时，她经常要洗澡，托莉为了满足她的任性，提水提得非常辛苦。她后来经常为此自责。  


神殿缺少神力，那里的自来水也需要人力。梅茵以为孤儿院也一样。戴莉娅带她去洗澡时，她胡乱冲了两下就想走，结果被戴莉娅拖了回来：“玛奥说了，洗澡一定要认真，不然会生病。”  


回到房间后，戴莉娅问她：“梅茵你会写字，出身应该不错，怎么会不爱洗澡呢？”她解释了以后，戴莉娅笑了起来：“这里的自来水用的是神力，不需要人提水。”梅茵愣了一会，然后跑出房间，冲向浴室。她要再洗一次！  


&nbsp;  


梅茵也遇到了困难，那就是玛奥为她安排的作业。  


那天晚饭后，玛奥给她做了一个小测验，结果表明她不需要上课。不过玛奥仍然要她和其他孩子一起行动：“……你可以坐在边上，不用听讲……”梅茵听到这里，想起了那件可怕的事情。  


玛奥拿出了一沓纸：“……你可以写这个，明天午饭时给我……”梅茵接过一看，第一张纸上写着几个字——简述你的国家的历史——她眼前一黑。玛奥又往纸上放了羽毛笔和墨水：“……别用炭笔了，不好保存。你现在就可以开始了，我等着看。”  


梅茵欲哭无泪。简述上下五千年，这种事情是人能干出来的么！  


&nbsp;  


梅茵考虑了一阵，然后决定按字面意义“简述”。她试着写了一下：“四千年前，一群人建立了……”她没有学过这里的历史，不知道该怎么表达“朝代”的概念，干脆替换成了“国家”：“……名为‘夏’的国家。四百年后，一群人击败了‘夏’国的军队，建立了的名为‘商’的国家……”  


梅茵很快就写到了战国。她计算了一下，这样“简述”的话，利用明天上课的时间就能写完。这时玛奥已经考校完了孩子们的功课，离开了孤儿院。梅茵毫不犹豫地扔下作业，加入了孩子们的游戏。  


&nbsp;  


第二天午饭时，梅茵有些忐忑地把“简述”交给了玛奥。玛奥看了一眼，声音温和：“我本来想让你看书的。”  


梅茵差点就要答应。但她想起了之前的事故，犹豫了起来——她知道自己看起来书来是什么样子——玛奥似乎看出了她的犹豫：“我会看着你，不会让你伤害别人。不过你要是再犯错，还是得挨鞭子。”  


挨鞭子不算什么，梅茵的回答力透纸背：“保证完成任务！”  


&nbsp;  


下午的劳动是清洗床单。梅茵洗不动床单，便帮着搬运。她一边把脏床单从房间里拿出来，一边回忆自己看过的历史书籍。这种简单劳动并不影响思考，她很快打好了腹稿。晚饭后，她开始书写燧人氏钻木取火的故事。  


戴莉娅凑了过来：“梅茵你在写什么？上午你就在写，还没写完么。”梅茵另找了张纸，写道：“这是我的读书券！”戴莉娅看得满头问号。梅茵没心思解释，就写道：“我要认真写，戴莉娅你不要和我说话好么？”戴莉娅答应了：“好吧。”  


梅茵埋头书写，从燧人氏写到神农氏，从黄帝大战蚩尤写到尧舜禅让，然后是大禹治水，废禅让，立世袭，夏朝建立……  


她这次写得很用心，还解释了“禅让”、“朝代”等概念。她写到商汤兴兵伐夏时，戴莉娅又跑了过来：“梅茵，洗澡去了。”  


梅茵抬头去看戴莉娅，眼神还有些迷茫，片刻后她醒过了神，写道：“第十钟响过了么？”戴莉娅道：“早就响了，我们得抓紧，不洗澡可是要挨鞭子的！”梅茵赶紧收拾起来。  


戴莉娅拿起一张纸看了起来：“你明明能写这么多字，为什么字迹还这么丑……”看着看着，她不说话了：“……这个后羿好狠，居然驱逐了君王！”  


这可能是戴莉娅第一次读故事，就算只是历史故事，而不是“后羿射日”这样的传奇，她依然读得津津有味。梅茵前世做了22年读者，现在突然成了作者，感觉颇为有趣。  


&nbsp;  


梅茵很快就有了第二个读者。她把作业交给玛奥后，玛奥看得十分认真。梅茵觉得这次应该没问题了。  


玛奥看完后又翻了一遍，然后说道：“今晚我会带书过来。”梅茵满足地闭上了眼睛，开始幻想晚上的幸福时光。玛奥又说道：“你要继续努力。”梅茵赶紧写字确认：“你还会让我看书吧？”玛奥道：“要像今天这样才可以。”  


梅茵不以为意，只要有书看，什么都好说。玛奥又说道：“后面的字越写越丑，你的手不行了吧，要不要休息一下？”梅茵写道：“休息时有书看么？”玛奥道：“没有。”梅茵写道：“那我不需要休息。”其实上午她的手就抽筋了，不过她觉得自己还能坚持。  


玛奥看了梅茵一眼，转向边上的戴莉娅。戴莉娅一直等在边上，热切地看着玛奥。玛奥看出了她的心思：“戴莉娅你也想看么？”戴莉娅连忙点头。玛奥道：“稍等。”  


梅茵以为玛奥要再翻阅一下。没想到玛奥看着桌上的纸笔说道：“御灵。”红光浮现，卷起了纸笔和墨水。羽毛笔蘸了墨水，飞快地在纸上书写起来。没过多久，梅茵的作业就被复制了一份。  


玛奥只复制了内容，字迹是他自己的，比梅茵的美观许多。他把复制品递给戴莉娅：“保存好，别丢了。”戴莉娅点头答应，接过了复制品。  


戴莉娅一点都不惊讶，似乎对此习以为常。梅茵则在心里感叹道：“好方便。”  


玛奥看向梅茵：“你看书的时间要缩减一半。”梅茵大惊失色。玛奥指着浮在空中的纸笔：“我要教你‘御灵’，这样你写起来会轻松一点。”梅茵写道：“我不要学！这和说好的不一样！”  


玛奥没有坚持：“那你继续用手写。不过要达到今天的量，我才会给你看书。”梅茵写道：“好。”  


&nbsp;  


晚上玛奥给梅茵带了一本书。梅茵兴高采烈地想去拿，玛奥却推开了她的手：“你拿不动。”她定睛看去，只见这本书大概相当于前世的八开本，又极为厚重，自己不可能拿得动。  


玛奥把书放到了桌上。梅茵扑了上去，看了一眼封面——《拥抱神力，踏上光荣的贵族之路》——她正准备翻开，玛奥又递给了她两本小册子，一本稍薄，一本稍厚。册子是手写的，梅茵认得是玛奥的字迹。她翻了一下，惊讶地发现这是两本“字典”。  


玛奥道：“这是我的朋友写的，他管这个叫‘字典’。”他指着稍薄的那本：“这里面是神力相关的词汇。”他又指向稍厚的那本：“这里面是古语词汇。”他最后指向那本书：“你会需要的。你应该知道怎么用，你的教材里有类似的内容。”  


梅茵点了点头，正要伸手翻书，又停了下来，看向玛奥。玛奥看出了她的想法：“我会看着你的。但你不能只想着依赖我，你还是要努力控制自己。”梅茵点头答应。  


“不能说话！不能说话！不能说话！”梅茵反复提醒了自己三次，然后翻开了书。  


&nbsp;  


这是一本介绍神力的书籍，梅茵相当感兴趣，读得津津有味。查字典分散了她的注意力，让她没有太过投入，戴莉娅叫她洗澡时，她立刻回过了神，没有再次失控。  


玛奥夸奖道：“有进步。”然后收起书和字典离开了。  


梅茵依依不舍地目送书本远去。戴莉娅似乎有些好奇：“那本书很有趣么？”梅茵点了点头。戴莉娅道：“我明天去找玛奥，让他给我也看看。对了……”戴莉娅晃晃手里的纸张：“……帝辛后来怎么样了？梅茵你又写了多少了？”  


梅茵摇了摇了头，表示自己还没写。这时她突然意识到了问题——今晚的时间都用来看书了，明天不可能完成“今天的量”！  


她扑向桌子，想赶紧写起来，结果被戴莉娅拖走了：“浴室要停水了！”洗完澡后，她写了一阵，然后被戴莉娅摁到了床上：“睡觉！”  


&nbsp;  


第二天上午，梅茵差点写断了手，还是没能完成“今天的量”。她交作业时心如死灰。玛奥看完后果然说道：“写得不错，但是少了点。明天继续。”梅茵写道：“今天的量可以累积到明天么？”玛奥道：“可以。”梅茵松了口气。  


戴莉娅今天也等在边上，玛奥照旧复制了一份。戴莉娅拿到复制品后又问道：“梅茵昨天看的那本书，可以给我也看一下么？”玛奥道：“好的，晚上我会把书带过来。你要记住，不可以给梅茵看。”  


梅茵按奈不住，写道：“为什么戴莉娅想看就能看，我就不可以！”玛奥举起作业：“因为你要写这个。”  


梅茵写道：“你这是……”她不知道怎么该表达‘剥削’：“……残害儿童！我的手都快写断了。”玛奥道：“你可以休息的。”梅茵无言以对。她想了一下，又写道：“那今晚可以教我御灵么？”  


晚上没书看了，她想趁机学习御灵，以后写起作业来会轻松一些。  


玛奥道：“你说过你不要学。”梅茵写道：“我改变主意了。”玛奥道：“那好。不过你拒绝了一次，那就不能只缩减一半的看书时间了。如果你要学，那么学会之前不准看书。”梅茵计算了一下，最终还是屈服了：“我要学。”  


&nbsp;  


晚上，梅茵愤愤地看着玛奥放下了书和字典，羡慕地看着戴莉娅翻开了书，然后惊愕地看着戴莉娅合上了书。  


“这不是和以前那些书一样么！这么难懂的书，梅茵你是怎么看下去的？”戴莉娅瞪大了眼睛。梅茵写道：“你不要看给我看！”戴莉娅道：“不行，玛奥说了不能给你看。”戴莉娅转向玛奥——玛奥在考校其他孩子的功课——说道：“玛奥，我不看了。”  


玛奥回答道：“放在那里就好。”他没有忘记补充：“不要给梅茵看。”  


梅茵咬紧牙关，写起了作业。
````
